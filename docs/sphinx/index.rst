******
Umpire
******

Umpire is a resource management libray that allows the discovery, provision,
and management of memory on next-generation hardware architectures.

- If you want to get and install Umpire, take a look at our getting started guide.
- If you are looking for developer documentation on a particular function,
  check out the code documentation.
- Want to contribute? Take a look at our developer and contribution guides.
